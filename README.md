# Flip-Flop and Counter Build

## Objective
To design and implement a D Flip-Flop and 4-bit Counter using Verilog HDL.

## Description
A D Flip-Flop stores one bit of data on the clock edge.
A Counter increments its value on each clock pulse.

## Software Used
- Verilog HDL
- ModelSim / Vivado

## Files Included
- flipflop_counter.v
- flipflop_counter_tb.v
- output.png

## Result
The D Flip-Flop and Counter were successfully designed and simulated.# FLIPFLOP-COUNTER-BUILD
